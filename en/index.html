Edition System

BigFive Updater version management

The first edition starting with 5 major package manager support. Ubuntu/Debian (APT), Fedora/RHEL (DNF), Arch (Pacman), openSUSE (Zypper), Alpine (APK) support. Codenames: Alpine, Beacon, Dream.

Fluent update experience. --doctor system health check, --history update history, --json structured output, --rollback rollback features. Codenames: Echo, Foxtrot.

Future edition targeting peak features. Plugin system, GUI integration, automatic update daemon and container support planned.

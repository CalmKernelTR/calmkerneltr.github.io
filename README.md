# calmkernel.tr

Personal portfolio and project showcase website.

**Live:** [calmkernel.tr](https://calmkernel.tr)

## Stack

- **Theme:** Aiero IT Services (index13) — Dark mode
- **Framework:** Bootstrap 5, Swiper, GSAP, ScrollTrigger, Lenis
- **Fonts:** Sora + Manrope (self-hosted WOFF2)
- **Hosting:** GitHub Pages (custom domain)
- **Languages:** Turkish (primary), English (/en/)

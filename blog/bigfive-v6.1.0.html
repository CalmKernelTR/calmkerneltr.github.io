üì¶ Quick Install

üéØ New Features

ü©∫ --doctor: System Health Check

Check your system's compatibility with BigFive using the new --doctor flag. Detailed analysis across 6 categories:

  • Required commands (curl, wget, gawk)
  • Package manager detection
  • Universal package managers (Flatpak, Snap)
  • System tools (fwupd, timeshift)
  • File system permissions
  • Disk space check
sudo guncel --doctor

üìú --history: Update History

View detailed history of the last 10 update operations:

  • Date and time
  • Executed backends
  • Success/failure status
  • Number of updated packages
guncel --history

üåç i18n: Turkish and English Support

BigFive now offers full Turkish and English language support: 

# Turkish
guncel --lang tr

# English
guncel --lang en

# Auto-detect from system locale
guncel

# CalmKernelTR.github.io - Repository Map

## Project Overview

**calmkerneltr.github.io** is the CalmKernel portfolio website hosted on GitHub Pages. It showcases open-source Linux system management tools and projects.

- **Domain**: calmkernel.tr
- **Visibility**: Public
- **Type**: Static Website (GitHub Pages)
- **Language**: HTML, CSS, JavaScript
- **Primary Language**: Turkish

## Design

- **Theme**: Dark mode (GitHub-inspired color scheme)
- **Colors**:
  - Background: #0d1117 (primary), #161b22 (secondary)
  - Accent: #3fb950 (green), #58a6ff (blue), #a371f7 (purple)
  - Text: #e6edf3 (primary), #8b949e (secondary)
- **Typography**: System fonts (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)

## Test Directory

The `test/` directory contains a premium HTML template being evaluated for future site redesign. It includes:
- 17 different homepage variations
- Various page templates (about, contact, blog, services, etc.)
- Complete asset library (CSS, JS, images, fonts)

**Note**: The test directory is for evaluation purposes only and not part of the live site.

## Development

```bash
# Local preview (use any static server)
python -m http.server 8000
# or
npx serve .

# Deploy (automatic via GitHub Pages on push to main)
git add -A && git commit -m "Update site" && git push
```

- HTTPS certificate managed by GitHub
